## Задание на лабораторную работу

Написать программу, на вход которой подается текстовый файл.
Входной текстовый файл должен быть разобран по словам, которые
необходимо поместить в хеш-таблицу. Программа должна уметь
запрашивать слова у пользователя и отвечать на вопрос: есть ли 
текущее слово в тексте или оно отсутствует
